02-02. 인라인 레벨 텍스트 테그

<strong><b> 굵게 표시

b 태그는 그냥 문서에서 굵게 표시하는 용

<em><i> 이탤릭체로 표시

<blockquote>랑 비슷하지만 그건 블록처리되고 들여쓰기까지 되버리지만 이건 인라인이라 한줄에 써진다.

css의 background-color속성으로 mark 태그의 배경색을 바꿀 수 있다고 한다.

03. 이미지와 하이퍼링크

03-01. 이미지

웹에서 사용하는 이미지 형식

웹에서 사용할 수 있는 이미지 파일 형식
파일 형식 설명
GIF (Graphic Interchange Form) 표시할 수 있는 색상 수가 최대 256가지지만 파일 크기가 작아서 아이콘이나 불릿 등 작은 이미지에 사용
투명한 배경이나 움직이는 이미지를 만들 수 있음
JPG/JPEG (Joint Photographic Experts Group) 사진을 위해 개발된 형식. GIF보다는 다양한 색상이나 명암 표현 가능
저장을 반복하다보면 화질이 떨어질 수 있다.
PNG (Portable Network Graphics) 투명 배경을 만들면서 다양한 색상을 표현할 수 있으며 네트워크용으로 개발되어 많이 사용

<img> 태그 - 이미지 삽입하기

src 속성을 사용해 이미지 파일이 있는 경로를 알려줘야 화면에 이미지 표시 가능

src 속성 - 이미지 파일 경로 지정
웹 문서의 파일의 위치를 기준으로 정해진다. 같은 경로면 파일 이름만 쓰면 된다.
하위 폴더면 폴더이름과 함께 작성한다.
웹 경로면 그 주소 그대로 쓰면 된다.

alt 속성 - 이미지를 설명해주는 대체 텍스트

화면낭독기가 시각 장애인들을 위해 읽어주는 텍스트
혹은 이미지가 제대로 표시되지 않을 때 해당 대체 텍스트가 표시되어 어떤 이미지인지 추측할 수 있다.
별 의미가 없는 경우에는 alt=""라고 지정해주면 된다

width, height 속성 - 이미지 크기 조정

속성을 사용하지 않으면 원본 사이즈로 나온다.
웹 서버에서 다운로드 하는 이미지 파일 용량은 그대로 두면서 사이트 디자인에 맞게 조정할 때 유용.
작은 이미지를 크게 하는 경우에는 화질이 나빠질 수 있다.

<figure>, <figcaption> 태그 - 이미지에 설명 글 붙이기

<figure>: 설명 글을 붙일 대상 지정
한 단위가 되는 요소들을 묶을 때 사용한다. 설명글을 붙이기 위해 묶고나서 <figcaption>을 쓰면 된다.

04-02. 사용자 입력을 위한 <input> 태그

<input> 태그 - 입력 항목 만들기

체크박스나 로그인버튼 같은것도 input 태그를 이용하고 폼 요소는 type 속성을 이용해 구분한다.

id 속성 사용하기
id를 지정하면 label로 캡션을 붙일 수도 있고 나중에 css로 다른 형태로 꾸밀수도 있음
최소 한개 이상의 문자에 공백은 없어야함.

<input>태그의 type 속성에서 사용가능한 유형

유형 설명
hidden 사용자에게는 보이지 않지만 서버로 넘겨지는 값을 가짐
text 한 줄짜리 텍스트를 입력할 수 있는 텍스트 상자를 넣는다
search 검색 상자를 넣는다.
tel 전화번호 입력 필드를 넣는다
url URL 주소를 입력할 수 있는 필드를 넣는다
email 메일 주소를 입력할 수 있는 필드를 넣는다
password 비밀번호를 입력할 수 있는 필드를 넣는다
datetime 국제 표준시(UTC)로 설정된 날짜와 시간(연, 월, 일, 시, 분, 초, 분할 초)를 넣는다
datetime-local 사용자가 있는 지역을 기준으로 날짜와 시간(연, 월, 일, 시, 분, 초, 분할 초)을 넣는다
date 사용자 지역을 기준으로 날짜(연, 월, 일)을 넣는다
month 사용자 지역을 기준으로 날짜(연, 월)을 넣는다
week 사용자 지역을 기준으로 날짜(연, 주)를 넣는다
time 사용자 지역을 기준으로 시간(시, 분, 초, 분할 초)를 넣는다
number 숫자를 조절할 수 있는 화살표를 넣는다
range 숫자를 조절할 수 있는 슬라이드 막대를 넣는다
color 색상 표를 넣는다
checkbox 주어진 항목에서 2개 이상 선택가능한 체크박스를 넣는다
radio 주어진 항목에서 1개만 선택할 수 있는 라디오 버튼을 넣는다
file 파일을 첨부할 수 있는 버튼을 넣는다
submit 서버 전송 버튼을 넣는다
image submit 버튼 대신 사용할 이미지를 넣는다
reset 리셋 버튼을 넣는다
button 버튼을 넣는다

type="hidden" - 히든 필드 만들기

화면상의 폼에는 보이지 않지만, 입력을 마치고 폼을 서버로 전송할 때 서버로 함께 전송되는 요소
<input type="hidden" name="이름" value="서버로 넘길 값">
name 속성으로 히든 필드의 이름을 지정하고 그에 대한 값은 value속성을 이용한다

type="text" - 텍스트 필드 만들기

<input type="text" [속성="속성 값"]>
한줄짜리 일반 텍스트.

텍스트 필드에서 사용할 수 있는 속성
속성 설명
name 텍스트 필드를 구별할 수 있도록 이름을 붙인다
size 텍스트 필드의 길이를 지정. 화면에 몇 글자까지 나오는 사이즈로 할 것인지 설정
value 텍스트 필드 요소가 화면에 표시될 때 텍스트 필드 부분에 표시될 내용. 디폴트 텍스트
maxlength 텍스트 필드에 입력할 수 있는 최대 문자 개수 지정

type="password" - 비밀번호 입력란

입력하는 내용을 * 등으로 표시하는 것. 속성은 text와 같지만 value속성만 없다.

type="search", type="url", type="email", type="tel" - 분화된 텍스트 필드

텍스트 필드와 비슷하지만 입력하는 정보에 맞게 분화된 텍스트 필드들이다.

type="search" - 검색상자

type="url" - URL 입력란
이 필드는 반드시 http://로 시작하는 사이트 주소를 입력해야한다.

type="email" - 메일주소 입력랸
메일 형식으로 입력해주는지 체크해준다

type="tel" - 전화번호 입력란
입력값을 체크하는 것보단 전화번호라는 것을 인식할 수 있게 한다. 이 값으로 바로 전화를 걸 수 있도록 한다.

type="number" - 숫자 입력

입력 내용을 숫자로 인식한다. 직접 입력이 기본이지만, 브라우저에 따라 위, 아래 표시의 스핀박스가 표시되기도 한다

type="range" - 슬라이드 막대로 숫자 지정

슬라이더가 표시된다.

type="number", type="range" 필드에서 사용할 수 있는 속성
속성 설명
min 필드에 입력할 수 있는 최소값을 지정한다. range일때 기본 최소값은 0
max 필드에 입력할 수 있는 최대값을 지정한다. range일때 기본 최대값 100
step 짝수나 홀수 등 특정 숫자로 제한하려고 할 때 숫자 간격을 지정. 기본값은 1이며 생략 가능
value 필드에 표시할 초기값

type="radio", type="checkbox" - 라디오 버튼과 체크박스 넣기

radio는 항목들 중 한개, checkbox는 항목들 두개 이상

라디오 버튼과 체크박스에서 사용할 수 있는 속성
속성 설명
name 라디오버튼이나 체크박스가 여러개 있을 때 서버에서 구분하기 위한 이름
value 선택한 라디오 버튼이나 체크박스를 서버로 알려줄 때 넘길 값 지정. 영문 또는 숫자. 필수 속성
checked 라디오 버튼들은 처음에 아무것도 선택되지 않은 상태로 표시되지만, 기본으로 선택해놓을 항목이 있으면 checked 속성을 사용한다

type="color" - 색상 선택 상자 표시

색상표에서 사용자가 색상을 선택할 수 있게 해준다. 색상 값은 16진수로 표시된다.

type="date", type="month", type="week" - 날짜 표시

type="time", type="datetime", type="datetime-local" - 시간 지정

날짜 뿐만 아니라 시간도 지정이 가능하다. 시간만 지정은 "time", 날짜도 지정은 "datetime" 또는 "datetime-local"

날짜나 시간 관련 유형 지정시 공통 사용 속성
속성 설명
min 날짜나 시간의 최소값
max 날짜나 시간의 최대값
step 스핀 박스의 화살표를 누를때마다 날짜나 시간을 얼마나 조절할지 지정
value 화면에 표시할 초기값. 시간은 00:00 ~ 23:59, datetime류는 날짜 다음에 T를 쓰고 24시간제로 지정한다.

type="submit", type="reset" - 서버 전송, 리셋 버튼

reset은 입력된 내용을 모두 지우거나 value에 있던 값으로 변경한다.
submit은 서버로 전송하는 것이다.

type="image" - 이미지 버튼 넣기

submit버튼 대신 전송 이미지를 넣을 수 있다.

type="button" - 버튼 넣기

submit이나 reset 같은 자체기능 없는 버튼. 동작은 스크립트 함수등을 연결해 사용한다.

type="file" - 파일 첨부

파일 첨부하는 버튼이 나오고 표시되는 내용은 브라우저마다 다르다

04-03. <input> 태그의 다양한 속성

autofocus 속성 - 입력 커서 표시하기

사용자가 포커싱하기 전에 먼저 포커싱해주는 기능.
참고로 아래는 포커스 기능을 넣었지만, 현재 페이지에 폼이 많아서 그런지 오토포커싱이 되지 않는다.

placeholder 속성 - 힌트 표시하기

언뜻 봐서는 value 속성처럼 미리 표시되어 있는 것 같지만, 선택하면 해당 값은 사라진다. 그리고 희미하게 표시되는 편이다.

readonly 속성 - 읽기 전용 필드 만들기

내용은 보여주되 입력을 할 수 없게 하는 필드. 삭제도 불가.
값은 boolean으로 "true", "false", "readonly"(true와 같다). 이후의 boolean들도 비슷한데 속성이름을 똑같이 넣으면 true인게 새롭다

required 속성 - 필수 필드 지정하기

서버로 submit하기전에 required 필드를 체크할때 검사할 수 있게 하는 속성. boolean

min, max, step 속성

해당 필드의 최소, 최대값, 해당 범위 내에서의 숫자 간격.
input 태그의 유형이 date, datetime, datetime-local, month, week, time, number, range인 경우만 가능

size, minlength, maxlength 속성 - 길이, 최소길이, 최대길이 속성

텍스트 길이 관련 조건.
size: 한줄짜리 텍스트와 관련된 필드에서 화면에 몇 글자까지 보여줄지
minlength: 최소 몇 글자 입력해야하는지
maxlength: 최대 몇 글자까지 입력할 수 있는지




그 외에 input 태그에서 사용할 수 있는 속성들
속성 설명
formaction 실행할 프로그램을 연결한다. 사용가능 유형: submit, image
formenctype 서버로 폼을 전송시 폼 데이터를 어떤 방식으로 해석할 것인지 지정. 사용가능 유형: submit, image
formmethod 서버로 폼을 전송하는 방식(get, post 등)을 지정한다. 이미 <form>태그 안에서 지정한 방식이 있어도 그 방식은 무시
formnovalidate 서버로 전송시 폼 데이터가 유효한지 여부를 표시하는 것. form에도 novalidate가 있는데 input에서도 표시
formtarget 폼 데이터를 서버로 전송한 후 서버의 응답을 어디에 표시할 것인지 타깃을 지정
height, width image 유형일때 이미지의 너비와 높이를 지정
list <datelist>에 미리 정의해놓은 옵션 값을 <input> 안에 나열하여 보여준다.
multiple 유형이 email, file 일때 두 개 이상의 값을 입력한다. 속성 이름만 표시하면 됨 (boolean?)

<select>, <optgroup>, <option> 태그 - 드롭다운 목록 만들기

드롭다운 목록은 <select>태그와 <option>태그를 이용해 표시한다.
시작은 select
내부에는 option. 여기에는 value 속성을 사용해 서버로 넘겨주기 위한 값을 지정한다

<select> 태그의 속성

  • 기본적으로 한가지 옵션이 표시됨
  • 화살표를 클릭하여 나머지 옵션을 살펴보고 필요한 한 가지 항목을 선택할 수 있다.
  • 이때, size 속성이나 multiple 속성을 이용하면 드롭다운 목록의 크기나 선택할 항목 갯수를 조정할 수 있다.
  • 속성 설명
    size 화면에 표시될 드롭다운 메뉴의 항목 개수를 지정한다.(크롬은 +1개 표시됨)
    multiple 여러개의 옵션이 함께 표시되면서 ctrl 키를 누른 상태로 드롭다운 메뉴에 있는 여러 항목을 선택할 수 있다.

<option> 태그의 속성

속성 설명
value 옵션 선택시 서버로 넘겨질 값을 지정
selected 화면에 표시될 대 기본으로 선택되어 있는 옵션을 지정
